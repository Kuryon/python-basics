# python-basics
## o_mnie.py
Program pyta się użytkownika o jego dane, po czym wypisuje je w print().

## zgadnij_liczbe.py
Plik ten jest grą, w której użytkownik musi zgadnąć liczbę w przedziale od 0 do 100. Gra podpowiada, czy jest blisko, czy daleko. Informuje również w ilu próbach liczba została odganięta i czy chce spróbować ponownie.

## kalkulator_bmi.py
Program sluży do kalkulowania poziomu stanu swojego BMI. Zawiera on 3 funkcje, w której ostatni wywołuje dwie pierwsze.
